创建一个新的待办事项：

  • 标题：待办事项的简短描述
  • 描述：待办事项的详细说明（可选）
  • 关联设备：选择要显示此待办事项的设备
  • 截止时间：设置待办事项的截止时间（可选）
  • 状态：可以选择直接标记为已完成

创建后，待办事项将发送到关联的设备，并可以在设备上标记完成状态。
